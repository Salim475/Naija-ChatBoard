# 🇳🇬 Naija ChatBoard

Naija ChatBoard is a modern community platform built for Nigerian conversations and communities. It provides real-time chat rooms, discussions, polls, notifications, and user profiles in a responsive and interactive interface.

## Tech Stack

* Next.js 16 (App Router)
* TypeScript
* Tailwind CSS
* Supabase
* PostgreSQL
* Supabase Realtime
* GSAP
* React

## Features

### Authentication

* User registration and login
* Protected routes
* Session management with Supabase

### Dashboard

* Modern responsive dashboard
* Interactive DotGrid background
* Mobile navigation and sidebar

### Real-Time Chat Rooms

* Multiple chat rooms
* Instant messaging with Supabase Realtime
* Live message updates

### User Profiles

* Profile pages
* Display names
* Bio and location
* Editable profile information

### Discussions

* Create discussions
* Reply to posts
* Real-time replies
* Automatic reply counters

### Poll System

* Create polls with multiple options
* One vote per user
* Live vote updates
* Automatic vote counting

### Notifications

* Real-time notification system
* Notification badge updates

## Project Structure

Built using the Next.js App Router architecture with TypeScript and Supabase backend services.
